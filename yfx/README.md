## 任务需求

1.智能捡球小车，路径规划，避障，识别，拾取

2.设计抓取的机械结构

3.YOLO图像识别3种球

## 方案计划

1.工作量体现在软件和机械结构上

2.采用四麦轮设计底盘

3.抓取机构采用滚轮扫入式，滚轮处采用传送带配合

4.视觉采用k210
