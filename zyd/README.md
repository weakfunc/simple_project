##  题目需求

1.垃圾清扫小车，路径规划，避障，自动识别，清理。

2.清理的方式：抓取，清扫。

3.YOLO图像识别垃圾，大体积垃圾抓取，小体积垃圾清扫。

4.识别大体积垃圾2种（纸盒，塑料瓶），小体积垃圾2种（纸屑，颜色液体）

## 方案计划

1.工作量体现在软件和机械结构上

2.采用两驱动轮履带式底盘

3.抓取采用2维舵机机械臂，清扫用小黄机

4.视觉用k210
